# Output Styles

Reusable markdown files that instruct Claude Code how to format its responses.
Select one with `/output-style <name>` or set a default in `settings.json`.

## Format

A plain `.md` file. The entire content is treated as instructions for Claude.

## Built-in styles

Claude Code ships with a few defaults (see docs). Files here extend/override
them.
